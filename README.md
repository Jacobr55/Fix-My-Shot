# Basketball Shot Analyzer

Basketball Shot Analyzer is a hackathon project built for **Hack K-State 2025**.

It’s a real-time AI web app that analyzes basketball shooting form using pose estimation and provides personalized feedback to help players improve.

## Tech Stack
- ASP.NET Core MVC  
- TensorFlow MoveNet (Pose Estimation)  
- SQLite + Entity Framework  
- ASP.NET Identity  
- Tailwind CSS
